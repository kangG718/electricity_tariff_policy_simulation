# electricity_tariff_policy_simulation
Industrial electricity usage data–based electricity tariff policy simulation framework that identifies VVIP power consumers and evaluates tariff adjustment impacts through data-driven analysis / 산업별 전력 사용 데이터를 기반으로 VVIP 전력 소비 집단을 식별하고 전기요금 단가 조정 정책의 영향을 데이터 기반 시뮬레이션으로 분석하는 프레임워크.
